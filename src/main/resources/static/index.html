Webhook Mocker

Webhook mocker

Post payload

curl -X POST https://webhook-mocker.herokuapp.com/webhook -H 'Content-Type: application/json' -d '{"key": {"innerKey": "aValue"}}'

Get payloads

curl -X GET https://webhook-mocker.herokuapp.com/webhook | jq

Delete payloads

curl -X DELETE https://webhook-mocker.herokuapp.com/webhook
